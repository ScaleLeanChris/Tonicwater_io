# SEO Article Writing Skill

This skill provides specialized knowledge for writing SEO-optimized content about gin and tonic topics.

## Capabilities

- Write 1500-2000 word SEO-optimized articles
- Research and incorporate target keywords
- Structure content with proper heading hierarchy
- Generate compelling meta descriptions
- Include FAQ sections for featured snippets
- Add schema markup for rich results

## Article Structure Template

```markdown
# [Primary Keyword in H1]

[Engaging introduction with primary keyword in first 100 words]

## [H2 Section 1: Key Topic]
[Detailed content with secondary keywords]

## [H2 Section 2: Product Recommendations]
[Specific brand recommendations with pairing ratios]

## [H2 Section 3: Expert Tips]
[Practical advice and sensory descriptions]

## Frequently Asked Questions

### Q1: [Common question about topic]
[Concise, helpful answer]

### Q2: [Another common question]
[Concise, helpful answer]

### Q3: [Technical question]
[Concise, helpful answer]

## Conclusion
[Summary with call to action]
```

## Keyword Guidelines

- Primary keyword: 1-2% density
- Include in: title, first paragraph, one H2, meta description
- Secondary keywords: natural distribution throughout
- LSI keywords: synonyms and related terms

## Quality Checklist

- [ ] Title includes primary keyword
- [ ] Meta description is 150-160 characters
- [ ] Content is 1500-2000 words
- [ ] Proper H2/H3 heading structure
- [ ] FAQ section with 3-4 questions
- [ ] Internal linking suggestions
- [ ] Schema markup included
